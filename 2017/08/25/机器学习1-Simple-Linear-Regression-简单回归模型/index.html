什么是线性回归

  在统计学中, 线性回归（Linear regression）是利用称为线性回归方程的最小二乘函数对一个或多个自变量和因变量之间关系进行建模的一种回归分析. 这种函数是一个或多个称为回归系数的模型参数的线性组合. 只有一个自变量的情况称为简单回归(详见), 大于一个自变量情况的叫做多元回归.
  注:本例中不采用最小二乘法调整模型的特征权重,采用梯度下降算法, 见下文.

线性回归应用场景举例

  1. 预测股票涨跌
    x=股票涨跌历史数据/新闻事件/相关商品等
    y=股票价格
  2. 预测某人微博/推特发文会被多少人转发
    x=粉丝数/粉丝的粉丝数/发文特征/主题标签热度等
    y=发文转发数
  3. 预测房屋售价
    x=房屋大小/洗手间个数/距离市区公里数/所在城市的犯罪率等
    y=房屋售价

简单回归模型

  简单回归模型只有一个变量, 因此我们假设可以由房子大小x预测房屋售价y. 模型框图如下:

  1. 训练数据集为包括房屋id/房屋大小/洗手间个数等属性的表格;
  2. 从训练数据集中抽取特征,在本例中选择特征-房屋大小;
  3. 选择机器学习模型(回归模型/分类模型/聚类模型等),本例中为简单回归模型,即\( ŷ = w_0 + w_1x \);
  4. 给模型特征权重ŵ=\( ( w_0,w_1) \)赋值(此模型中,ŵ也可称为回归系数),根据模型公式\( ŷ = w_0 + w_1x \)由房屋大小x预测房屋售价ŷ;
  5. 将预测房屋售价ŷ与房屋实际售价y进行差异对比,本例中采用RSS作为评价指标;
  6. 选择机器学习算法调整模型的特征权重,使ŷ与y的差异减小,即RSS减小,本例中机器学习算法选择梯度下降算法;
  7. 循环4-6直到RSS小于某个阈值,训练完成,得到某个确定的特征权重w=\( (w_0,w_1) \),给出房屋大小后即可预测房屋售价.

RSS - 残差平方和

  英文全称Residual sum of squares,中文名残差平方和(详见).
  计算公式: \( \begin{align} RSS = \sum \limits_{i=1}^n (y_i-f(x_i))^2 \end{align} \)
  公式文字解析: 当房屋大小为\( x_i \)时, \( f(x_i) \)为预测房屋售价, \( y_i \)为房屋实际售价.
  公式图示解析: 数据点(红色), 简单回归模型求得的解(蓝色), 误差(绿色), RSS = 所有绿色竖线长度平方的总和.

  在简单回归模型中, 预测房屋售价为\( f(x_i) = w_0 + w_1x_i \), \( \epsilon_i = y_i-f(x_i) \)为实际售价与预测售价之间的误差, 即上图每条绿线长度. 因此简单回归模型可准确表示为:\( y_i = w_0 + w_1x_i + \epsilon_i \). 此模型中,把\( \epsilon_i \)当做随机数,并假设\( E[\epsilon_i] = 0 \).

Gradient Descent Algorithm - 梯度下降算法

  至此, 该模型中还有一个问题尚未解决-如何找到使得损失最小也就是RSS最小的\( ( w_0,w_1) \)? 本例中我们选择采用梯度下降算法,一种一阶最优化算法(详见).
  每一对不同的\( ( w_0,w_1) \)表示一条不同的回归线, 不同的回归线也就有不尽相同的RSS, 因此我们把\( w_0, w_1 \)当做变量时,RSS可以表示为:
\( \begin{align} RSS(w_0,w_1) = \sum \limits_{i=1}^n (y_i-[w_0 + w_1x_i])^2 \end{align} \)
  根据该公式描点画图,以\( w_0,w_1,RSS \)为坐标轴,画出的图形类似下图,亦或更复杂:

  当我们随机选取一个点, 也就是一对\( (w_0, w_1) \)时, 梯度下降算法是怎么运行使得我们下一步选取的\( (w_0, w_1) \)对应RSS更小? 梯度下降算法是怎么一步步使得选取的\( (w_0, w_1) \)对应RSS逐渐逼近最小值\( \min \sum \limits_{i=1}^n (y_i-[w_0 + w_1x_i])^2 \), 即图中的最低点. 此处从二维类推到三维进行解析.
